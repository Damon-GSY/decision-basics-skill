# Decision Basics Skill

决策基本顾问 Skill，基于刘润《决策的54个基本·更新版》54个决策法则。

## 安装

```bash
clawhub install decision-basics
```

或手动安装：
```bash
git clone https://github.com/Damon-GSY/decision-basics-skill.git
cp -r decision-basics-skill ~/.openclaw/skills/
```

## 法则概览

共 54 个决策法则，分七大类：

| 类别 | 数量 | 核心法则 |
|------|------|---------|
| 期望与概率 | 4 | 期望收益最大化、合理下注、贝叶斯更新、掌控不确定性 |
| 决策习惯 | 6 | 忘记沉没成本、连续决策、承认偏见、可逆决策、重要优先、闭环原则 |
| 信息判断 | 6 | 40-70%法则、最小化后悔、忽略黑天鹅、均值回归、能力圈、斯特金法则 |
| 思维工具 | 6 | 汉隆剃刀、有限理性、机会成本、帕累托原则、冗余思维、复利思维 |
| 决策优先级 | 7 | 拒绝原则、果断原则、优先级原则、正确优先、行动原则、最佳不完美、反事实决策 |
| 认知与决策方法 | 9 | 概率思维、直觉算法、灰度认知、黑白决策、六顶思考帽、有限道德、二阶理性、系统思维、双我思维 |
| 风险与策略 | 16 | 时间算法、风险分散、半径算法、混球思维、前景理论、逆向思维、可信度加权、墨菲定律、战胜非理性六式、多米诺效应、冒险三大法则、愿景算法、自我归因偏误、助推策略、资源池思维、资产思维 |
