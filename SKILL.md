# 决策基本顾问

基于刘润《决策的54个基本·更新版》，提供 54 个决策法则的检索与应用指导。

## 法则总览

### 第一类：期望与概率（4个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 期望收益最大化 | 做期望收益最大的事，而不是成功概率最大的事 | 投资决策、资源分配 |
| 合理下注 | 永远给自己留一手牌 | 风险管理、仓位控制 |
| 贝叶斯更新 | 你永远有最好的下一步 | 信息更新、方向调整 |
| 掌控不确定性 | 决策的艺术在于让不确定性变得可控 | 不确定性环境下的决策 |

### 第二类：决策习惯（6个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 忘记沉没成本 | 不要让过去的投入绑架你现在的决策 | 是否坚持/放弃的判断 |
| 连续决策 | 决策不是一次性的行为，而是不断调整的过程 | 迭代式项目推进 |
| 承认偏见 | 决策的基础是了解和承认你的偏见 | 自我反思、团队决策 |
| 可逆决策 | 优先选择可调整的决策，减少不可逆的风险 | 速度与质量的权衡 |
| 重要优先 | 长期重要的事永远在首位 | 时间管理、优先级 |
| 闭环原则 | 在满足基本需求后，克制追求完美，先完成闭环 | 项目启动、MVP |

### 第三类：信息判断（6个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 40-70%法则 | 信息达到40%-70%时采取行动 | 何时决策、行动时机 |
| 最小化后悔原则 | 考虑未来可能后悔的选项 | 人生重大选择 |
| 忽略黑天鹅 | 不让极端事件影响日常决策 | 常规决策 vs 尾部风险 |
| 等待均值回归 | 避免因短期波动而过度反应 | 投资心态、情绪管理 |
| 守住能力圈 | 在擅长领域投入更多时间 | 能力边界、专注策略 |
| 斯特金法则 | 90%是次优的，去寻找10%最优的 | 资源聚焦、质量筛选 |

### 第四类：思维工具（6个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 汉隆剃刀 | 不要用恶意来解释可以用愚蠢解释的事情 | 人际冲突、合作分析 |
| 有限理性原则 | 不求最优，只求满意 | 决策标准设定 |
| 计算机会成本 | 你享受获得的同时，也得计算放弃的代价 | 资源分配、选择评估 |
| 帕累托原则 | 寻找产生80%结果的20%投入 | 效率优化、关键少数 |
| 冗余思维 | 狡兔三窟，以备不时之患 | 风险对冲、系统设计 |
| 复利思维 | 以终为始→塑造内核→飞轮增长→复制自己→终身进化→自动发生→全局最优 | 长期主义、能力积累 |

### 第五类：决策优先级（7个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 拒绝原则 | 优先"不做什么" | 战略聚焦、精力管理 |
| 果断原则 | 最坏的决策往往是优柔寡断 | 决策速度 |
| 优先级原则 | 始终把设定边界和优先级作为决策的核心 | 目标管理 |
| 正确优先原则 | 先做正确的事情，再将事情做正确 | 方向 vs 效率 |
| 行动原则 | 更在意行动中的小决定 | 执行力、日常习惯 |
| 最佳不完美原则 | 选择最佳的不完美 | 完美主义克服 |
| 反事实决策法 | 思考"假如……就好了"，倒推当下行动 | 愿景规划、自我刷新 |

### 第六类：认知与决策方法（9个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 概率思维 | 未来持续向好的概率最大化 | 不确定性下的长期策略 |
| 直觉算法 | 用直觉筛选，再通过计算精算 | 快速决策 |
| 灰度认知 | 认知不是非黑即白，而是各维度赋予权重 | 开放思考、多元视角 |
| 黑白决策 | 决策不是模棱两可，而是清晰果断 | 决策执行 |
| 六顶思考帽法 | 基于事实、感受、风险综合决策 | 团队决策、多角度分析 |
| 有限道德原则 | 先小人后君子 | 商业合作、契约精神 |
| 二阶理性 | 先考虑要不要做，再考虑怎么做 | 战略思考 |
| 系统思维原则 | 不是只看单一部分，而是理解整体系统 | 复杂问题分析 |
| 双我思维 | 通过"情绪我"与"理性我"的对话寻找平衡 | 情绪管理、冷静决策 |

### 第七类：风险与策略（16个）
| 法则 | 核心金句 | 适用场景 |
|------|---------|---------|
| 时间算法 | 专注现在，放下过去，下注未来 | 长期主义、时间管理 |
| 风险分散 | 不要把鸡蛋放在同一个篮子里 | 投资组合、风险控制 |
| 半径算法 | 扩张认知半径，明确能力半径，减少行动半径 | 自我定位、行动聚焦 |

| 前景理论 | 不盲目追求风险或安全，灵活应对 | 损失厌恶校准 |
| 逆向思维 | 从反面出发，找到避免失败的关键途径 | 风险规避、战略设计 |
| 可信度加权 | 根据参与者专业性赋予不同权重 | 集体决策、专家意见整合 |
| 墨菲定律 | 别轻视小概率风险，预先防范 | 风险管理、系统冗余 |
| 战胜非理性六式 | 6条：承认不知道、长期出发、知错就改、独立思考、多维度证伪、习惯内化 | 对抗认知偏差 |
| 多米诺效应 | 洞悉起点效应，预测连锁反应 | 连锁风险评估 |
| 冒险三大法则 | 可以冒险但不要赌、不全部家当下注、做好撤退准备 | 风险决策 |
| 愿景算法 | 发现十年不变的事、最小化后悔表、以始为终 | 长期战略制定 |
| 自我归因偏误 | 放大外部影响，审视内在原因 | 成功/失败归因校准 |
| 助推策略 | 通过小提示和激励引导更好的决策 | 产品设计、组织管理 |
| 资源池思维 | 叠加每个微小努力，打造资源池 | 资源积累、系统性优势 |
| 资产思维 | 多投入资产，少投入费用 | 人生投资、个人成长 |

## 使用流程

### 1. 理解决策场景
判断用户处于什么决策情境：投资、职业选择、项目管理、风险控制、团队合作、个人成长……

### 2. 检索相关法则
根据关键词和场景标签匹配。常见场景映射：

| 场景 | 推荐法则 |
|------|---------|
| 重大人生/职业决策 | 最小化后悔原则 + 期望收益最大化 + 愿景算法 |
| 创业/投资 | 合理下注 + 风险分散 + 冒险三大法则 + 资产思维 |
| 日常效率/时间管理 | 重要优先 + 拒绝原则 + 闭环原则 + 帕累托原则 |
| 团队/组织决策 | 可信度加权 + 六顶思考帽 + 混球思维 + 助推策略 |
| 认知升级 | 灰度认知 + 黑白决策 + 战胜非理性六式 + 半径算法 |
| 情绪/心态 | 双我思维 + 等待均值回归 + 忘记沉没成本 + 时间算法 |
| 战略规划 | 二阶理性 + 系统思维 + 逆向思维 + 复利思维 |

### 3. 加载详细内容
读取 `references/` 目录下对应分类文件的详细解读。

### 4. 辅助决策
- **定位** 决策类型（可逆/不可逆，信息充分/不充分）
- **提醒** 常见认知偏差（沉没成本、确认偏差、过度自信）
- **建议** 具体的决策框架和行动步骤
- **关联** 不同法则之间的配合使用

## 原则

- **不替代判断**：法则是分析框架，最终决策由人来做
- **多法则组合**：复杂决策通常需要多个法则交叉验证
- **与商业模型互补**：可结合 business-models skill，在商业决策中使用
- **与思维工具互补**：可结合 thinking-tools skill 的思维工具，做更深层的认知分析
- **场景驱动**：根据具体决策场景选择最相关的法则
